# Stoat
A Sable datapack with new physics properties for modded and vanilla blocks!
Some notable ones being Sky Lanterns from The Block Box floating away and Architect's Palette Abyssaline acting as a ballast!

Requires Sable to work, Create Simulated and the mods this tags blocks from are optional but recommended.

You may include and modify this datapack for use in modpacks and videos as long as you link back to either the modrinth or this github page. Additionally, if any of the authors of Sable or the mods tagged want to incorporate the tags or custom physics properties into their own mods, they are free to do so. You may use this as a reference and inspiration for your own datapacks. If you'd like to suggest support for certain mods, please bring it up in the issues, or if you know how to implement said support yourself, please make a pull request or your own datapack/mod! (Weasel would be a GREAT name for one.)

Please do not reupload this to other sites.
