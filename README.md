Stoat is a datapack for <a href="https://modrinth.com/mod/sable">Sable</a> with new physics properties for modded and vanilla blocks!
Some notable ones being Sky Lanterns from <a href="https://modrinth.com/mod/the-block-box">The Block Box</a> floating away and <a href="https://modrinth.com/mod/architects-palette">Architect's Palette</a> Abyssaline acting as a ballast!

Requires Sable to work, <a href="https://modrinth.com/mod/create-aeronautics">Create Simulated</a> and the mods this tags features from are optional but recommended.

You may include and modify this datapack for use in modpacks and videos as long as you link back to either the <a href="https://modrinth.com/datapack/stoat">modrinth</a> or github page. Additionally, if any of the authors of Sable or the mods tagged want to incorporate the tags or custom physics properties into their own mods, they are free to do so. You may use this as a reference and inspiration for your own datapacks. If you'd like to suggest support for certain mods, please bring it up in the issues, or if you know how to implement said support yourself, please make a pull request or your own datapack/mod! (Weasel would be a GREAT name for one.)

Please do not reupload this to other sites.
